Today drones working in transportation. Let's plan efficient drone movement.

The input value is a list of integers. One or more number of this represent the existence of the package to be transported and its destination You have to return the shortest moving distance for the drone to complete all transportation as an integer.

  • Note:
    • The drone is initially at position 0 (most left).
    • The drone can carry only one package at a time.
    • You can only drop a package on destination.
    • The drone finally returns to position 0.

Examples: 

delivery_drone([0, 2, 0]) == 4
delivery_drone([0, 0, 1, 2]) == 6

Input: The information of packages (a list of integer).

Output: The shortest moving distance (an integer).

Preconditions:

  • 3 ≤ len(input) ≤ 30
  • 1 ≤ number of packages ≤ 9
